# AlleenBijDeVijver
#### (AloneAtThePond)
Changes a channel's name based on the amount of people in it.  
Keep in mind that Discord only lets you update a channel once every 5 minutes.

## Docker deployment
Dockerfile in AlleenBijDeVijver folder. Envvars:
- BOT_TOKEN
- GUILD_ID
- CHANNEL_ID
- NAME_0
- NAME_1
- NAME_M

All are required.
